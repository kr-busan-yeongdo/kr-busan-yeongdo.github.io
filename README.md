# 영도구 · 부산광역시

> 영도구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-yeongdo.github.io/**

영도구는 대한민국 자치구 가운데 유일하게 하나의 섬만으로 이루어진 부산의 자치구로, 부산항과 남항 사이에 떠 있는 영도(옛 절영도)가 구역 전체를 이룹니다. 옛 이름 절영도(絶影島)는 이곳 목장에서 기른 말이 그림자(影)가 끊어져(絶) 보일 만큼 빠르다는 데서 유래해, 예로부터 명마의 산지로 이름났습니다. 봉래산이 섬 한가운데 우뚝 솟아 있고 그 자락을 따라 계단식 마을과 조선소, 어항이 촘촘히 들어차 있어 바다와 뭍이 뒤엉킨 독특한 경관을 자아냅니다. 태종대와 흰여울문화마을, 국립해양박물관, 깡깡이예술마을 등 바다를 배경으로 한 명소가 밀집해 부산을 대표하는 해양 관광지로 손꼽힙니다. 근대 조선산업의 발상지이자 한국전쟁 피란민의 애환이 서린 영도대교의 무대로, 부산 근현대사의 상징 공간으로 남아 있습니다.
